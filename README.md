Страница 1: Вычисление функции «a»

Математическое выражение:
a = ln(y^(-√|x|)) · (x - y/2) + sin²(arctg(z))

Входные параметры:

x (вещественное число)

y (вещественное число, обязательное условие: y > 0)

z (вещественное число)

Критическое ограничение: Для обеспечения корректности вычисления натурального логарифма необходимо строгое соблюдение условия y > 0.

Страница 2: Кусочная функция «e»

Логика вычисления:
Функция рассчитывается по трем вариантам в зависимости от четности параметра i и знака x:

e = {
i√f(x), если i нечетное и x > 0
i/2·√|f(x)|, если i четное и x < 0
√|if(x)|, в остальных случаях
}

Вариативность f(x): Пользователь может выбрать одну из трех функций с помощью интерфейса (радиокнопки):

sh(x) — гиперболический синус
x² — квадрат числа
e^x — экспонента
Входные данные: x (вещественное), i (целое, определяет четность).

Страница 3: Табуляция функции «y» и визуализация

Аналитическое выражение:
y = x⁴ + cos(2 + x³ - b)

Исходные данные для цикла:

Начальное значение (x₀) = 4.6

Конечное значение (xₖ) = 5.8

Величина приращения (dx) = 0.2

Константа (b) = 1.3

Реализуемый функционал:

Построение ряда значений функции путем итерации аргумента x от начального до конечного значения с заданным шагом.

Отображение результатов в виде таблицы пар (x, y).

Построение графика функции для наглядного представления данных.
